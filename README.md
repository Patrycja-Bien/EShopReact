# React App Run from File Version

## How to run
### `git clone https://github.com/Patrycja-Bien/Patrycja_Bien_Web_Wroclaw.git`
### `git checkout main`
### `npm install react`
### `npm start`
### Navigate through pages on `http://localhost:3000/`
